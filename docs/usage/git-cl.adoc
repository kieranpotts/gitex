= `git cl`

Shortcut for `git clone`. All parameters are forwarded to the underlying `git clone` command.

== Usage

----
$ git cl [...]
----

== Examples

Clone a repository:

----
$ git cl https://github.com/<user>/<repo>.git
----

Clone into a specific directory:

----
$ git cl https://github.com/<user>/<repo>.git <dir>
----

Clone with options (shallow clone):

----
$ git cl --depth 1 https://github.com/<user>/<repo>.git
----

Clone a specific branch:

----
$ git cl --branch develop https://github.com/<user>/<repo>.git
----

See https://git-scm.com/docs/git-clone[git-clone] for further documentation.
